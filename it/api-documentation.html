Illimitato leads API

Autenticazione

Tutte le richieste API devono includere un'intestazione di autorizzazione . L'intestazione deve fare riferimento alla tua chiave privata.

Quando ti registri per l'accesso a API, Unlimited leads assegna una chiave privata API al tuo account. La tua chiave privata API è nota solo a te e a Unlimited leads. È importante mantenere entrambe riservate per proteggere il tuo account. La tua chiave API è accessibile dal link Account.

API Chiama: trova email

POST https://api.unlimited-leads-to.online

Restituisce un oggetto codificato in JSON con l'indirizzo email della persona e del dominio specificati. Le chiamate a API vengono misurate e fatturate se viene restituita un'email con il risultato. Se non è disponibile alcuna email, non verrà addebitato alcun costo.

Parametri della richiesta POST

  • firstName - Il nome della persona (obbligatorio)
  • lastName - Il cognome della persona (obbligatorio)
  • dominio - Il dominio da interrogare (obbligatorio)

Parametri di richiesta di esempio

  • Nome - John
  • cognome - Doe
  • dominio - company.com

Risposta POST

{ "email": "john.doe@company.com" }

Esempio di richiesta curl

> curl -vk \ -H "Content-Type: application/json" \ -H "Authorization: api_private_key" \ -X POST "https://api.unlimited-leads-to.online" \ -d '{ "firstName": "John", "lastName": "Doe", "domain": "company.com", }'

Nessuna risposta e-mail trovata

{ "email": null }
