Comment installer l'extension Chrome Apollo en quelques clics

L'installation de l' extension Chrome Apollo est un processus rapide qui prend moins d'une minute. Cette extension, plébiscitée par plus de 400 000 utilisateurs et dotée d'excellentes évaluations, est essentielle pour tout commercial souhaitant maximiser son efficacité en prospection B2B.

Étape 1 : Trouver l'extension dans le Chrome Web Store

Commencez par effectuer une recherche Google pour "Apollo Chrome extension" ou accédez directement au Chrome Web Store. L'extension officielle est rédigée "Apollo.io - Trouvez gratuitement les coordonnées B2B".

Étape 2 : Ajouter l'extension à Chrome

Cliquez sur le bouton "Ajouter à Chrome" puis confirmez en cliquant sur "Ajouter l'extension" dans la boîte de dialogue qui s'affiche. L'installation démarre automatiquement et ne prend que quelques secondes.

Étape 3 : Accéder à la documentation

Après l'installation, vous serez redirigé vers un article de la base de connaissances Apollo qui explique en détail toutes les fonctionnalités de l'extension. Il est recommandé de consulter cette documentation pour exploiter pleinement l'outil.

L' extension Chrome Apollo offre un ensemble complet de fonctionnalités qui permettent aux commerciales de prospection B2B d'accéder aux informations cruciales sans quitter leur navigation. Voici les principales fonctionnalités qui font de cette extension un outil incontournable.

Accès instantané aux coordonnées

Visualisez immédiatement les coordonnées professionnelles de vos prospects, y compris leurs emails et numéros de téléphone, directement depuis leur profil LinkedIn ou autre site web.

Apollon partout

Intégration avec les séquences

Ajoutez directement des prospects à vos séquences d'emails sans quitter votre navigation, optimisant ainsi votre flux de travail commercial.

Synchronisation avec votre compte Apollo

L'extension se synchronise parfaitement avec votre compte Apollo principal, ce qui signifie que toutes les actions effectuées via l'extension (ajout de contacts, notes, tâches) sont immédiatement disponibles dans la plateforme Apollo. Cette intégration garantit une expérience utilisateur fluide et une gestion centralisée de vos activités de prospection.

Comment utiliser l'extension Apollo avec LinkedIn

L'intégration de l' extension Chrome Apollo avec LinkedIn constitue l'un des cas d'utilisation les plus puissants pour les professionnels de la vente. Cette combinaison vous permet de transformer LinkedIn, déjà riche en informations professionnelles, en un véritable outil de prospection commerciale ultra-efficace.

Exploration des profils LinkedIn avec Apollo

Lorsque vous naviguez sur LinkedIn avec l'extension Apollo activée, vous remarquerez une icône jaune sur le côté gauche de votre écran. En cliquant sur cette icône, un panneau s'ouvre et affiche automatiquement des informations détaillées sur le profil que vous consultez.

Par exemple, lorsque vous visitez le profil d'un prospect, l'extension affiche :

  • Coordonnées professionnelles : email et numéro de téléphone (si disponibles)
  • Informations sur l'entreprise : secteur, taille, description des services
  • Technologies utilisées : stack technologique que l'entreprise utilise
  • Historique de financement : données sur les levées de fonds si applicables
  • Offres d'emploi actifs : postes que l'entreprise cherche à pourvoir

Ajout de prospects à des séquences depuis LinkedIn

L'une des fonctionnalités les plus pratiques de l'extension est la possibilité d'ajouter directement des prospects à vos séquences d'emails sans quitter LinkedIn. Voici comment procéder :

  1. Visitez le profil LinkedIn du prospect
  2. Ouvrez l'extension Apollo en cliquant sur l'icône jaune
  3. Cliquez sur "Add to Sequence" dans le panneau de l'extension
  4. Sélectionnez la séquence appropriée dans la liste déroulante
  5. Confirmez l'ajout pour démarrer automatiquement la séquence d'emails

Recherche de contacts similaires

L'extension Apollo vous permet également d'identifier rapidement d'autres contacts pertinents dans la même entreprise ou occupant des postes similaires dans d'autres organisations. Cette fonctionnalité est particulièrement utile pour élargir votre liste de prospects qualifiés à partir d'un contact initial prometteur.

Apollo Everywhere : accédez aux données sur n'importe quel site

"Apollo Everywhere" est une fonctionnalité révolutionnaire de l' extension Chrome Apollo qui vous permet d'accéder aux données d'Apollo non seulement sur LinkedIn, mais sur n'importe quel site web d'entreprise que vous visitez. Cette capacité transforme littéralement chaque page web professionnelle en opportunité de prospection.

Comment activer Apollo Everywhere

L'activation d'Apollo Everywhere est simple :

  1. Cliquez sur l'icône de l'extension Apollo dans votre barre d'outils Chrome
  2. Dans le menu qui s'affiche, activez l'option "Apollo Everywhere"
  3. L'extension est maintenant prête à analyser n'importe quel site web d'entreprise

Exploitation des données sur les sites d'entreprise

Une fois Apollo Everywhere activé, chaque fois que vous visitez un site web d'entreprise, vous pouvez cliquer sur l'icône Apollo pour faire apparaître un panneau affiché :

  • Vue d'ensemble de l'entreprise : secteur, taille, chiffre d'affaires estimé
  • Technologies identifiées : outils et plateformes utilisées par l'entreprise
  • Informations sur le financement : historique de levées de fonds
  • Employés clés : liste des décideurs et leurs coordonnées

Questions fréquentes sur l'extension Chrome Apollo

L'extension Chrome Apollo est-elle gratuite?

Oui, l' extension Chrome Apollo est disponible gratuitement. Cependant, certaines fonctionnalités avancées ont avancé un abonnement à Apollo.io. La version gratuite vous permet déjà d'accéder aux informations de la base des entreprises et à un nombre limité de contacts chaque mois. Pour une utilisation intensive, un abonnement payant à Apollo est recommandé. Pour compléter votre stratégie avec des données de qualité, visitez Unlimited Leads .

Comment l'extension Apollo se synchronise-t-elle avec mon compte Apollo.io?

Après installation, l'extension vous demandera de vous connecter avec vos identifiants Apollo.io. Une fois connecté, toutes les actions effectuées via l'extension (comme l'ajout de contacts à des séquences ou la consultation d'informations) sont automatiquement synchronisées avec votre compte principal Apollo. Cette synchronisation est bidirectionnelle : les modifications apportées dans la plateforme sont également visibles dans l'extension. Pour optimiser votre prospection sur Apollo, découvrez nos exemples d'emails de prospection efficaces .

L'extension Chrome Apollo fonctionne-t-elle sur d'autres navigateurs que Chrome?

Bien que conçue principalement pour Google Chrome, l'extension Apollo est également disponible pour Microsoft Edge, qui est basée sur Chromium. Pour Mozilla Firefox, il n'existe pas de version officielle à ce jour. Si vous utilisez Safari ou un autre navigateur, vous devrez passer par Chrome ou Edge pour bénéficier de cette extension. Découvrez comment tirer le meilleur parti de l'extension Apollo quel que soit votre navigateur.

Quelles sont les limites de l’extension Chrome Apollo ?

Les principales limites de l' extension Chrome Apollo sont liées à votre forfait Apollo.io. Avec un compte gratuit, vous aurez un nombre limité de crédits mensuels pour consulter des emails et numéros de téléphone. De plus, certaines fonctionnalités avancées comme l'ajout à des séquences complexes ou l'accès à des filtres de recherche avancé avec un abonnement payant. L'extension peut également connaître des diminutions lors de l'analyse de sites web très complexes. Pour compléter vos besoins en données, découvrez nos bases de données B2B qualifiées .

Comment résoudre les problèmes courants avec l'extension Chrome Apollo?

En cas de problème avec l' extension Chrome Apollo , essayez ces solutions : 1) Déconnectez-vous puis reconnectez-vous à l'extension, 2) Désactivez puis réactivez l'extension dans Chrome, 3) Videz le cache de votre navigateur, 4) Désinstallez puis réinstallez l'extension, 5) Vérifiez que vous utilisez la dernière version de Chrome. Si le problème persiste, consultez le centre d'aide Apollo ou contactez leur support. Pour une alternative fiable en matière de données de prospection, explorez les services d'Unlimited Leads .
