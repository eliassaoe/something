Invalid LinkedIn URL Format

The LinkedIn URL you entered is not supported.

We only accept LinkedIn profile URLs that contain readable usernames in the format:

✅ Supported URL formats:

https://www.linkedin.com/in/john-doe-123/
https://www.linkedin.com/in/jane-smith-marketing/
https://linkedin.com/in/eliasse-hamour-08194821a/

❌ Unsupported URL formats:

https://www.linkedin.com/in/fiuyngiuhefyg8379249/
https://www.linkedin.com/in/ACoAABXYZ123/
https://www.linkedin.com/in/abc123def456/

Why this restriction?

URLs with encrypted or random IDs cannot be reliably processed by our scraping system. We need URLs with readable usernames to ensure accurate data extraction.

How to find the correct URL:

Visit the LinkedIn profile directly and copy the URL from your browser's address bar. Make sure it contains the person's name or a readable username.
