Fragen & Antworten zum Email Permutator

Wie verwende ich den E-Mail-Permutator?

Die Verwendung unseres E-Mail-Permutators ist einfach:

  1. Geben Sie den Vornamen der Person in das Feld „Vorname“ ein
  2. Geben Sie den Nachnamen in das Feld „Nachname“ ein
  3. Geben Sie im Feld „Domäne“ die Firmendomäne (ohne „http://“ oder „www.“) ein.
  4. Klicken Sie auf „E-Mail-Adressen suchen“, um alle möglichen Kombinationen zu generieren
  5. Kopieren Sie einzelne E-Mails oder verwenden Sie „Alle kopieren“, um die vollständige Liste zu erhalten
  6. Klicken Sie optional auf „In Gmail öffnen“, um mit dem Verfassen an diese Adressen zu beginnen.
Wie finde ich die E-Mail-Adresse einer Person?

Um die E-Mail-Adresse einer Person herauszufinden, sind mehrere Strategien erforderlich:

  1. Nutzen Sie unseren E-Mail-Permutator: Generieren Sie alle möglichen Kombinationen basierend auf Ihrem Namen und Ihrer Firmendomäne
  2. Testen Sie die Adressen: Verwenden Sie unseren E-Mail-Validator oder senden Sie Test-E-Mails, um zu überprüfen, welche Adresse funktioniert
  3. Überprüfen Sie die Website des Unternehmens: Manchmal werden E-Mail-Formate auf Kontaktseiten oder in Pressemitteilungen erwähnt
  4. Suchen Sie die E-Mail eines Kollegen: Wenn Sie das E-Mail-Format einer anderen Person im selben Unternehmen kennen, verwendet die Zielperson wahrscheinlich dasselbe Format
  5. Soziale Medien: Überprüfen Sie ihre LinkedIn-, Twitter- oder anderen Profile, da sie manchmal E-Mail-Adressen auflisten

Unser E-Mail-Permutator beschleunigt diesen Vorgang erheblich, indem er alle möglichen Formate automatisch generiert.

Was tun, wenn Sie die E-Mail-Adresse einer Person gefunden haben?

Wenn Sie die richtige E-Mail-Adresse gefunden haben, gehen Sie wie folgt vor:

  1. Überprüfen Sie es: Verwenden Sie unser E-Mail-Validierungstool, um zu bestätigen, dass die Adresse gültig ist
  2. Verfassen Sie eine personalisierte Nachricht: Allgemeine Massen-E-Mails werden ignoriert – passen Sie Ihre Nachricht an den Empfänger an
  3. Verwenden Sie eine klare Betreffzeile: Machen Sie sie relevant und überzeugend (probieren Sie unser Tool zum Generieren von Betreffzeilen aus).
  4. Halten Sie es prägnant: Respektieren Sie ihre Zeit mit einer kurzen, fokussierten Nachricht
  5. Fügen Sie einen konkreten Call-to-Action hinzu: Geben Sie klar an, was Sie als Nächstes tun möchten
  6. Nachfassen: Wenn Sie keine Antwort erhalten, senden Sie nach 3-5 Tagen eine höfliche Nachfassaktion

Denken Sie daran, dass das Herausfinden der E-Mail-Adresse einer Person nur der erste Schritt ist. Inhalt und Zeitpunkt Ihrer Nachricht sind für die Erlangung einer Antwort gleichermaßen wichtig.

Was unseren E-Mail-Permutator anders macht

Im Gegensatz zum einfachen Raten von E-Mail-Adressen verwendet unser E-Mail-Permutator ausgeklügelte Algorithmen, um alle möglichen Kombinationen basierend auf gängigen E-Mail-Mustern zu erstellen, die von Unternehmen weltweit verwendet werden. Das Tool generiert Variationen, darunter:

  • Vornamenvarianten: john@company.com, j@company.com
  • Nachnamenvarianten: smith@company.com, s@company.com
  • Kombinierte Namensformate: john.smith@company.com, johnsmith@company.com
  • Anfangskombinationen: js@company.com, j.smith@company.com
  • Trennungsvarianten: john_smith@company.com, john-smith@company.com

Mit über 34 verschiedenen, sofort generierten E-Mail-Mustern erhöhen Sie Ihre Chancen, beim ersten Versuch die richtige E-Mail-Adresse zu finden, erheblich.

Best Practices zum Suchen von E-Mail-Adressen

  1. Beginnen Sie mit dem E-Mail-Muster des Unternehmens: Wenn Sie die E-Mails anderer Mitarbeiter des Unternehmens kennen, suchen Sie nach konsistenten Mustern (vorname.nachname@firma.com, vornameinitialnachname@firma.com usw.).
  2. Verwenden Sie Überprüfungstechniken: Überprüfen Sie nach dem Generieren von Permutationen mithilfe von E-Mail-Überprüfungstools oder durch Senden einer Testnachricht, welche Adresse gültig ist.
  3. Überprüfen Sie die E-Mail-Richtlinien des Unternehmens: Große Unternehmen befolgen häufig einheitliche Konventionen für die Benennung von E-Mails für alle Mitarbeiter.
  4. Probieren Sie alternative Domänen aus: Einige Unternehmen verwenden unterschiedliche Domänen für unterschiedliche Abteilungen oder Regionen (z. B. sales.company.com vs. company.com).
  5. Verwendung mit LinkedIn: Suchen Sie den vollständigen beruflichen Namen der Person auf LinkedIn und verwenden Sie dann unseren Permutator mit der Unternehmensdomäne.
