Unlimited Leads API

Simple, powerful APIs for email finding, verification, LinkedIn scraping, and database export

Authentication

Include your API key in the Authorization header for all requests:

Authorization: Bearer YOUR_API_KEY

Get your API key from your account dashboard.

Email Finder

POST https://unlimited-finder-api.hamoureliasse.workers.dev

Find professional email addresses using a person's name and company domain.

Parameters

Parameter Type Status Description
firstName string REQUIRED Person's first name
lastName string REQUIRED Person's last name
domain string REQUIRED Company domain (e.g., company.com)

Example Request

curl -X POST https://unlimited-finder-api.hamoureliasse.workers.dev \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "firstName": "John", "lastName": "Doe", "domain": "company.com" }'

Response ‚úì Email Found

{ "email": "john.doe@company.com" }

Response ‚úó Not Found

Database Export

POST https://api-unlimited-leads-export.hamoureliasse.workers.dev/

Export leads from the database with advanced filtering options. Maximum 5,000 records per request.

Parameters

Parameter Type Status Description
action string REQUIRED Must be "export"
limit number REQUIRED Number of records (1-5000)
searchParams.country string REQUIRED Target country (see allowed values below)
searchParams.industry string OPTIONAL Industry filter
searchParams.company string OPTIONAL Company name filter
searchParams.keywords string OPTIONAL Keywords to search
searchParams.city string OPTIONAL City filter
searchParams.jobtitle string OPTIONAL Job title filter
searchParams.companysize string OPTIONAL Company size: 1-25, 26-50, 51-250, 251-1000, 1000+
searchParams.hasemail string OPTIONAL Filter by email: "yes" or "no"
